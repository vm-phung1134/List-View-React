# List View ReactJs

## Run Locally

### 1. Clone repo

```
$ git clone https://github.com/vm-phung1134/List-View-React.git
$ cd list-view
```
### 2. Run
```
$ npm run start
```
### 3. Open browser
Run this on browser: [http:localhost:3000](http://localhost:3000)
